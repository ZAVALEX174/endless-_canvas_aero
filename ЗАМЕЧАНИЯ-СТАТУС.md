- Глобальная `sealedNodes` (Set) хранит ключи запечатанных тупиков (`main5.js`)
- Солвер: `isSinkNode()` и `isTrueDeadEnd()` проверяют `sealedNodes` (`airSolver.js`)
- Правый клик рядом с тупиком → пункт меню «Запечатать/Открыть тупик» (`contextMenu.js`)
- Красный квадрат-маркер на запечатанных тупиках (`visualization.js`)
- Сохранение/восстановление `sealedNodes` в JSON и автосохранении (`projectManager.js`)

### Детали фиксов п.9-10 (airSolver.js):
- **BFS seeding** (стр.527): `sourceFlow > 0` -> `sourceFlow !== 0` — реверсные вентиляторы теперь корни BFS
- **Pressure init** (стр.110-125): дифференцированная инициализация давления (-1/0/+1 по знаку sourceFlow)
- **Route loss** (стр.271): `sourceFlow > 0` -> `sourceFlow !== 0` — метрики сети для реверса
- **topoOrder** (стр.706): пересчёт после солвера (солвер переориентирует рёбра)
- **KCL-диагностика** (стр.694-707): проверка баланса в узлах (пропуск тупиков)
