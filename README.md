# Калькулятор вентиляционной сети шахты

Веб-приложение для расчёта распределения воздуха в шахтной вентиляционной сети.
Интерактивное рисование схемы на canvas + автоматический расчёт потоков, давлений и депрессии.

## Быстрый старт

Открыть `index.html` в браузере. Бэкенд не требуется, сборка не требуется.

## Стек

- HTML5 + CSS3 + Fabric.js (canvas)
- Vanilla JS, метод узловых потенциалов (Гаусс-Зейдель)
- Модульная архитектура: IIFE-паттерн, экспорт в `window`

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `L` | Режим рисования линии |
| `G` | Показать/скрыть сетку |
| `S` | Разделить линии по пересечениям |
| `A` | Вкл/выкл автоматическое разделение |
| `Alt+N` | Заблокировать/разблокировать узлы |
| `Alt+P` | Запустить расчёт воздухораспределения |
| `Delete` | Удалить выбранный объект (блокируется в заблокированном узле) |
| `Shift+Delete` | Принудительно удалить объект (обходит блокировку узла) |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Повторить |
| `Ctrl+S` | Сохранить чертёж в файл |
| `Ctrl+O` | Загрузить чертёж из файла |
| `Escape` | Отмена текущего режима / закрытие модалок |
| `Alt` (зажат) | Привязка линии к краю объекта |

## Автосохранение

Схема автоматически сохраняется в localStorage каждые 2 секунды после последнего изменения. При перезагрузке страницы появляется баннер с предложением восстановить последнюю схему или начать заново.

## Кнопка "Разделить по центрам"

Кнопка `splitAllLinesAtObjectCenters` проходит по всем объектам-изображениям на схеме (вентиляторы, клапаны, атмосфера и т.д.) и разрезает линии, проходящие через их центры. Это нужно, чтобы расчётный движок видел узел в точке расположения объекта.

**Примечание:** при включённом `autoSplitMode` (по умолчанию) линии разрезаются автоматически в момент добавления объекта (`splitLinesAtImagePosition`). Кнопка полезна только если autoSplit был отключён в момент размещения объектов и нужно разрезать все линии задним числом.

## Экспорт

- **JSON** — полное сохранение/загрузка проекта
- **CSV** — таблица ветвей с расчётными данными
- **PDF** — схема + таблица ветвей
